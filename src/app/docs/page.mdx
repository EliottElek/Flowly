
# **Flowly**  

**Flowly** is a lightweight, modern Kanban app designed to streamline task organization and team productivity.  
Built with **Supabase** and **Next.js 15**, it provides an intuitive interface and robust backend features to manage your projects efficiently.

## ğŸ’» **Getting Started**

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)  
- [Supabase CLI](https://supabase.com/docs/guides/cli)  
- [Docker](https://www.docker.com/) (optional, for local Supabase setup)  

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Flowly.git
   cd Flowly
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure your `.env` file for Supabase:
   ```bash
   NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
   ```

4. Start the app:
   ```bash
   npm run dev
   ```
